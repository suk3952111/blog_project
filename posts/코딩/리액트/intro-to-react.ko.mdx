# React 소개

React는 사용자 인터페이스를 구축하기 위한 선언적, 효율적, 그리고 유연한 JavaScript 라이브러리입니다. 이 글에서는 React의 기본 개념에 대해 알아보겠습니다.

## JSX 소개

JSX는 JavaScript를 확장한 문법으로, React와 함께 사용하여 UI 구조를 직관적으로 설명할 수 있습니다.

```jsx
function Hello(props) {
  return <div>Hello, {props.name}</div>;
}
```

## 컴포넌트와 Props

React에서 컴포넌트는 UI의 한 조각입니다. Props를 통해 컴포넌트에 데이터를 전달할 수 있습니다.

```jsx
function Welcome(props) {
  return <h1>Hello, {props.name}</h1>;
}
```
