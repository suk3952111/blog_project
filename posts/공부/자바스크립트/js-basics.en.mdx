# JavaScript Basics

JavaScript is a programming language for making web pages interactive.

## Variables and Data Types

In JavaScript, you can use various data types.

```js
let name = 'John Doe';
let age = 25;
let isStudent = true;
```

## Functions

Functions are one way to reuse code.

```js
function greet(name) {
  return 'Hello, ' + name;
}
```
