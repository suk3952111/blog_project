# 자바스크립트 기초

자바스크립트는 웹 페이지를 동적으로 만들기 위한 프로그래밍 언어입니다.

## 변수와 데이터 타입

자바스크립트에서는 다양한 데이터 타입을 사용할 수 있습니다.

```js
let name = '홍길동';
let age = 25;
let isStudent = true;
```

## 함수

함수는 코드를 재사용하기 위한 방법 중 하나입니다.

```js
function greet(name) {
  return '안녕, ' + name;
}
```
