# CSS 기초

CSS는 웹 페이지의 스타일을 정의하기 위해 사용되는 언어입니다.

## CSS 선택자

CSS 선택자를 사용하여 HTML 요소를 선택하고 스타일을 적용할 수 있습니다.

```css
p {
  color: blue;
}
```

## 박스 모델

모든 HTML 요소는 박스 모델로 표현됩니다. 이 모델은 마진, 테두리, 패딩, 그리고 실제 콘텐츠를 포함합니다.

```css
.box {
  margin: 10px;
  border: 1px solid black;
  padding: 20px;
}
```
